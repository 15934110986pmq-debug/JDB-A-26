### 补丁 P0-b：Bootstrap → 明确 i.i.d. 高斯前提，删除"非参等价 HAC"过度声称

> 上一轮我曾声称"采用更稳健的非参 Bootstrap，绕过 HAC 修正得到等价覆盖率"——经核代码 `q2_solve.py:127`，**实际为参数化 Gaussian Bootstrap**（每次 `rng.normal(0, σ̂)` 抽取），不是非参的。已重写为：

```
严格修正路径应采用 Newey-West / HAC 异方差自相关一致估计，或 block bootstrap
（保留相关结构的非参重采样）。本文实现的是参数化 Gaussian Bootstrap，
继承 i.i.d. 高斯假设，因此既不应对自相关也不应对厚尾，不能等价代替 HAC；
它的实际作用是用 Monte Carlo 量化非线性优化器与数据形状的有限样本效应。
Bootstrap CI 略小于 CRB（0.40 vs 0.65 s）的可能成因不唯一：
  (a) 参数化方法在轻微非高斯/相关下偏乐观
  (b) CRB 是 Hessian 二阶展开，多模/平坦区可能高估
  (c) B=80 重采样数较小本身带来估计方差
两者保持同量级已可作为"互核"，不应硬解读为单一原因。
```

### E. 集体盲点扫查（collective-blind-spot sweep）

> 二审最关键的环节：所有一审者都漏掉了什么？

请重点考察以下方向：

- **E.1 临界值的语境**：除了 $\chi^2(2)$ 的三档区分，论文中还有什么"教科书数字"被自动套用而未核语境？例如：
  - Anderson-Darling 临界值 0.752（针对 unmodified 还是 modified $A^2$？样本量校正了吗？）
  - Shapiro-Wilk 在 $N>5000$ 上的有效性？
  - KS 检验在估计参数后的分布扰动（要用 Lilliefors 修正）？
- **E.2 同源性**：除了 $J^* = 2(\hat\sigma^2)$ 外，还有别的"看似独立验证、实则同源"的吗？例如残差均值 $\bar r \approx 0$ 是否就是 LS 一阶必要条件的代数后果？
- **E.3 估计的渐近假设**：CRB 与 $\sigma_{\Delta t}$ 公式中的渐近性，对我们的 $N$ 量级是否真成立？或者其实有限样本偏差占主导？
- **E.4 参数耦合**：$\Delta t$、$\Delta x$、$\Delta y$ 联合估计时，三者协方差矩阵的非对角项有多大？我们报告的边际 CI 是否低估了联合不确定度？
- **E.5 周期性歧义的"局部稳定"**：Bootstrap CI 1.5s ≪ 周期 628s 这个论证，是否假设了 Bootstrap 不会跨候选解跳跃？如果 Bootstrap 中某些迭代落到 C2 候选（差一个周期），CI 就会爆炸——这种情况实际发生过吗？
