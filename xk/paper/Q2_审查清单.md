## B. 数值结果一致性

1. 给定 $\hat\sigma_x = 0.730, \hat\sigma_y = 0.697$ m，
   ```
   E[J*] = 2(0.730² + 0.697²) = 2(0.5329 + 0.4858) = 2.0374 m²
   ```
   实测 $J^* = 2.038$。**99.97% 吻合**意味着什么？是否过拟合？

2. **σ_Δt 量级核算**：粗略 $\sigma_{\Delta t} \sim \sigma / \bar v$，代入 $\sigma = 0.71$ m, $\bar v \approx 3.7$ m/s：
   $$\sigma_{\Delta t} \approx 0.19\,\mathrm s$$
   实测 Cramér-Rao $0.65$ s, Bootstrap $0.40$ s。这一差距合理吗？(提示：考虑有效样本量、轨迹形状对 Fisher 信息的贡献)

3. **KF 后验方差**：单路 $\sigma^2 = 0.50$，等权融合 $\sigma^2/2 = 0.25$，KF 0.025。从理论上预测 KF 应当达到的方差量级（提示：$N_{\text{eff}}$ 估计、运动学先验贡献）。
