# A 题：多源融合机器人定位及任务优化

## 背景

在自主移动机器人、无人车等智能装备的导航与定位任务中，单一传感器定位方式易受环境干扰、自身硬件限制等因素影响，存在定位精度不足、鲁棒性差、数据频率单一等问题，难以满足复杂场景下机器人实时高精度定位的工程需求。实际应用中，通常采用多源异构传感器组合定位方案，通过融合不同原理、不同频率的定位数据，实现定位精度、稳定性与实时性的协同提升。

现有机器人实际定位场景中的两类核心定位数据：

- **定位方式 1**：输出频率 4 Hz，采样周期 0.25 s
- **定位方式 2**：输出频率 5 Hz，采样周期 0.20 s

两类定位数据由不同采集设备获取，存在**起始时间不同步**、**采样频率不同**、**数据含随机噪声**等问题，直接融合会导致位置信息错位、轨迹失真，无法输出高频率、高精度的连续定位结果。

请根据上述背景和附件数据，完成以下建模任务。

---

## 问题 1（附件 1，无噪声、仅时间偏差）

附件 1 两类数据在采集过程中**无噪声**影响，但存在**设备开机的先后不同**。请：

1. 建立两类数据的**时间对齐模型**；
2. 给出两种方式的**时间偏差** $\Delta t$；
3. 输出 **10 Hz 的位置轨迹**。

## 问题 2（附件 2，含噪声 + 系统偏差）

附件 2 两类数据存在**随机测量噪声**和**固定的系统偏差**。请：

1. 建立**数据对齐与融合模型**；
2. 给出两种方式的**时间偏差** $\Delta t$ 与**系统偏差** $(\Delta x,\Delta y)$；
3. 输出 **10 Hz 的位置轨迹**。

## 问题 3（附件 3，实测数据）

附件 3 两类数据是**实际测量数据**。请：

1. **判断**是否存在系统偏差（含统计/可视化证据）；
2. 在此基础上完成**问题 2 的任务**（时间偏差、系统偏差、10 Hz 轨迹）。

## 问题 4（附件 3 + 附件 4，任务调度优化）

机器人按照**附件 3 中的轨迹**运动，在运动过程中需要对沿途目标点执行 **"模拟射击"** 或 **"拍照扫描"** 任务，目标点坐标见 `附件4.xlsx`（射击目标 S01–S18 共 18 个，拍照目标 P01–P18 共 18 个）。

请对任务目标进行**优化设计**（约束条件见附录），**尽可能多地完成**射击和拍照任务，并将结果填入 `result.xlsx`（不改变附件中的红色文字内容）。

## 附录：任务约束

### （1）射击任务约束

设射击时刻机器人与目标间的距离为 $d$，机器人线速度 $v$，加速度 $a$：

| 项 | 约束 |
|---|---|
| 单次射击命中率 | $85\%$ |
| 射击距离 | $d \in [5\,\mathrm{m},\ 30\,\mathrm{m}]$ |
| 线速度 | $v \le 2\,\mathrm{m/s}$ |
| 加速度幅值 | $\lvert a\rvert \le 1.5\,\mathrm{m/s^2}$ |
| 瞄准校准时间 | $1.5\,\mathrm{s}$，**校准期间上述约束需持续满足** |

### （2）拍照任务约束

| 项 | 约束 |
|---|---|
| 同一目标的不同拍照角度 | 要求**尽量多角度**拍摄 |
| 拍照距离 | $d \in [10\,\mathrm{m},\ 40\,\mathrm{m}]$ |
| 不同角度间方向角差异 | $\ge 60^{\circ}$ |
| 线速度 | $v \le 1.5\,\mathrm{m/s}$ |
| 加速度幅值 | $\lvert a\rvert \le 1.5\,\mathrm{m/s^2}$ |
| 相机对准时间 | $0.5\,\mathrm{s}$，**对准期间上述约束需持续满足** |

> 提示：约束区间与速度上限均通过解析 docx 内 MathType 公式对象 (`word/media/image*.wmf` 的可见字符串) 还原得到，与原题一致。
